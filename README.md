# Temka Client - Advanced Minecraft Cheat Client

Профессиональный веб-сайт с системой аккаунтов.

## Установка и запуск

### Вариант 1: Локальный режим (без сервера)

Просто откройи `index.html` в браузере. Всё работает!

**Преимущества:**
- ✅ Работает сразу
- ✅ Не нужна установка
- ✅ Идеально для демо

**Недостатки:**
- ❌ Данные хранятся только в браузере
- ❌ Теряются при очистке кэша

### Вариант 2: Серверный режим (с Node.js)

**Требования:**
- Node.js (v14 или выше)
- npm

**Шаги:**

1. **Установите зависимости:**
```bash
npm install
```

2. **Запустите сервер:**
```bash
npm start
```

3. **Откройте браузер:**
```
http://localhost:3000
```

4. **Измените API в index.html:**
Замени в `index.html`:
```html
<!-- Было -->
<script src="api-local.js"></script>

<!-- Стало -->
<script src="api.js"></script>
```

## Архитектура

### Локальный режим
- **api-local.js** - API в браузере
- **localStorage** - хранилище данных

### Серверный режим
- **server.js** - Express сервер
- **temka_users.db** - SQLite база данных

### Frontend
- **index.html** - главная страница
- **app.js** - логика приложения
- **download.js** - функции скачивания
